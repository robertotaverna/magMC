Monte Carlo code for magnetar persistent emission.
If you use this code in a publication, please cite
R. Taverna, R. Turolla, V. Suleimanov, A.Y. Potekhin, S. Zane, MNRAS, 492, 5057 (2020)

The code driver is Magnetars.f90

If you want to reproduce blackbody or condensed surface emission
select the corresponding modules.

Select your input parameters in the Mag.inp file.

For condensed surface emission the surface temperature
must be set also in the Constants module.
